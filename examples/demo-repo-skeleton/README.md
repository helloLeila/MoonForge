# Demo Repo Skeleton

This repo skeleton is intentionally minimal.

Use it with `scripts/run_demo.sh` to verify:

1. `moonforge init`
2. `moonforge task`
3. `moonforge check`
4. `moonforge report`

Key demo paths mirror the design document:

- `frontend/src/pages/UserList/index.tsx`
- `frontend/src/components/DepartmentFilter.tsx`
- `backend/src/main/java/com/acme/common/security/AuthFilter.java`
- `backend/src/main/java/com/acme/user/service/high-risk/UserAuditService.java`
